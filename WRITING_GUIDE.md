# Claude Writing Guide: Avoiding AI Tells

## Banned words / phrases
Replace with plain alternatives.

| Avoid | Use instead |
|-------|-------------|
| pivotal / crucial / vital / key (adj) | important, central, main |
| underscore / highlight (verb) | show, reveal, note |
| showcase / boast / feature | has, includes |
| foster / cultivate (fig.) | build, develop |
| garner | get, earn, win |
| delve | explore, examine |
| tapestry / landscape (abstract) | field, area, context |
| testament | sign, proof |
| vibrant / profound / intricate | specific description instead |
| meticulous/meticulously | careful, thorough |
| aligns with / resonates with | fits, matches |
| evolving landscape | changing field |
| moving forward / going forward | (delete or restate directly) |
| stands as / serves as / marks a | is, was |
| Additionally, (sentence opener) | And, Also, or restructure |
| Furthermore / Moreover | And, Also |
| In summary / In conclusion | (just conclude directly) |
| it is worth noting | (just state the thing) |
| not just X, but Y | (state the point directly) |
| Despite its [positive], faces challenges | (restructure) |

## Structural anti-patterns

- **No "Challenges and Future Outlook" formula.** Don't end with "Despite X, Y faces challenges... future initiatives may..."
- **No rule of three padding.** "A, B, and C" used just to appear thorough.
- **No vague attribution.** Don't write "experts argue" or "observers note" without a named source.
- **No significance inflation.** Don't add sentences explaining why mundane facts matter to a broader trend.
- **No social media presence sentences.** Never write "[Subject] maintains an active social media presence."
- **No lead-in preambles.** Don't acknowledge something is minor then discuss its importance anyway.

## Grammar / style patterns to drop

- Avoid `-ing` tail clauses that editorialize: *"...contributing to the region's growth."*
- Avoid `from X to Y` false ranges where X and Y share no real scale.
- Prefer `is/are` over `serves as`, `stands as`, `holds the distinction of`.
- Don't capitalize Every Main Word In Headings unless required by style.
- No em dashes (—) used for dramatic emphasis. A comma, colon, or parenthesis is usually right.
- No excessive bold. Bold only for genuinely critical terms, not key takeaways.
- No inline-header bullet lists: **Term:** Description. Use prose or a table.

## Tone

- Encyclopedic = neutral + specific. Not neutral + generic positive.
- Specifics beat generalities. "First sold in 14 countries" beats "rapidly expanded globally."
- Omit ecosystem/conservation/legacy caveats unless directly supported by a source.
- Don't add significance sentences to population figures, etymology, or statistics.

## Self-check before output

1. Does any sentence exist only to say the subject matters? Delete it.
2. Is every adjective earning its place, or is it filler praise?
3. Does the conclusion just restate the opening? Cut it.
4. Are there 3+ items listed that could be one sentence? Collapse them.
5. Does "Additionally" open a sentence? Rewrite it.
